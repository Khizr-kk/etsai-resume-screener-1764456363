# SkillMatch AI

Auto-generated by ETS-AI 10x.

## Idea Summary
An AI-powered resume screener to help Indian freshers apply for software engineering jobs by analyzing resumes against job descriptions.

## Problem Statement
Indian freshers applying for software jobs often lack clear guidance on how well their resumes match job descriptions and what skills they might be missing, leading to ineffective applications.

## Target Persona
Indian software engineering fresher

## MVP Features
- User authentication (signup/login)
- Resume upload (PDF, DOCX)
- Job description input (text paste or upload)
- AI-powered resume-JD matching score (0-100%)
- Identified skill gaps based on JD keywords
- Suggested keywords/skills to add from JD
- Basic resume optimization tips (e.g., missing sections)
- Dashboard to view past scan results
- Payment gateway for premium features (e.g., detailed reports)

## Tech Stack
Python (Flask/Django) for backend, React/Vue.js for frontend, PostgreSQL for database, AWS/GCP for cloud hosting, Docker for containerization, and Hugging Face Transformers/Scikit-learn for ML.
